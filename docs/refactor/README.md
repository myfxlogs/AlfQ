## 使用规则

1. **执行 Agent**：开始任何 PR 之前必须全文阅读 v1.0 文档
2. **变更管理**：方案变更必须新增版本号文档（不修改已发布版本），并在本 README 顶部更新「当前生效方案」
3. **冲突仲裁**：当本目录与 `docs/16/17/19/26/27` 出现冲突，以 `docs/26 §3,§4` 锁定的版本与豁免规则**优先**，重构方案不得绕开
4. **完成归档**：重构全部 PR 合并后，将 v1.0 文档移至 `docs/refactor/archived/` 并在本 README 注明完成时间
